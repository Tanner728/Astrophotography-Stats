# Astrophotography-Stats
The purpose of this repo is to house code for creating graphs and charts showing various statisics about my astrophotography work. We will see how this goes :)


## Goals 
The idea is to create a python program or programs, henceforth refered to as the program, to easily create a data dashboard for use on a personal website or anywhere else that you might want to display your astrophotography statistics and images. The program should be able to read both FITS and XISF files as they are largely the same in regards to how they are used in this context and are commonly used in the astrophotography space.
